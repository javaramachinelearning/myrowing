앱 사용법

설정법 2 (Setting)

앱에 연동 정보 등록하기

획득한 인증 키와 ID를 앱에 입력하면, 그 순간부터 나만의 강력한 맞춤형 로잉 대시보드가 완성됩니다.

설정 모달 열기 및 등록

  • 대시보드 메인 화면 우측 상단의 톱니바퀴(⚙️) 아이콘을 클릭하여 설정 모달을 엽니다.
  • Concept2 Access Token: 발급받은 긴 영문/숫자 조합의 토큰을 붙여넣습니다.
  • User ID: 본인의 Concept2 고유 숫자 ID (예: 2XXXXX)를 입력합니다.
  • Gemini API Key (선택): AI 분석을 앱 안에서 바로 확인하려면 입력합니다.
  • 정보 세이브 버튼을 누르고 [실시간 로그북 동기화]를 클릭해 통신 테스트를 진행합니다.

💡 부가 설명 (철저한 개인정보 보호)

보안 걱정은 하지 않으셔도 됩니다. 입력하신 모든 연동 데이터(API Key, ID 등)는 오직 사용자 본인의 브라우저 내부(Local Storage)에만 암호화되어 저장되며, 그 어떤 외부 제3자 서버나 데이터베이스로도 수집되거나 전송되지 않습니다.
