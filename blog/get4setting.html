앱 사용법

설정법 1 (Get4Setting)

필수 연동 Key 및 ID 획득하기

이 분석 앱을 사용하려면 Concept2 공식 로그북에서 본인의 데이터를 가져올 수 있는 '열쇠(API Key)'가 필요합니다.

1. Concept2 API Key & ID (필수)

  • Concept2 Logbook 사이트에 로그인합니다.
  • 우측 상단 프로필 > Edit Profile로 들어가면 상단 URL(주소창)이나 프로필 페이지 내에 본인의 고유 User ID (예: 2708646)가 보입니다. 이를 복사합니다.
  • 다시 프로필 메뉴에서 Applications (또는 Developer/API) 메뉴로 진입하여 새로운 Personal Access Token을 생성하고 발급된 긴 문자열을 복사합니다.

2. Gemini AI API Key (선택)

앱 내에서 바로 코칭 피드백을 보고 싶다면 구글 AI 스튜디오에서 무료 API 키를 받을 수 있습니다.

  • Google AI Studio에 접속합니다.
  • 새 프로젝트를 만들고 Create API Key 버튼을 눌러 키를 발급받습니다.

💡 부가 설명 (자동 클립보드 복사 병행)

구글 API 키 발급이 번거롭다면 비워두셔도 무방합니다. 앱 하단의 '✨ Gemini 링크'를 누르면, 앱이 내부적으로 스트로크 데이터가 포함된 풍성한 프롬프트를 클립보드에 자동 생성 및 복사해 줍니다. 복사된 내용을 제미나이 화면에 붙여넣기만 하세요!
